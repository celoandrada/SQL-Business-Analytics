# Advanced SQL

## Purpose

This section contains more advanced SQL queries used for business analytics and reporting.

Topics practiced include:

- Multi-table joins
- Aggregation with SUM()
- GROUP BY and HAVING
- Subqueries
- NOT IN
- UNION
- INTERSECT
- Category-level averages
- Ranking logic
- Business sales analysis
