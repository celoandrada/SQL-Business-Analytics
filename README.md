# SQL Business Analytics

This repository contains SQL coursework and database projects focused on business analytics, relational database design, and analytical querying.

The work covers foundational SQL concepts through more advanced database operations used in business environments.

---

## Topics Covered

### Basic SQL Queries

Concepts practiced:

- SELECT
- WHERE
- ORDER BY
- DISTINCT
- LIKE
- COUNT()
- SUM()
- JOIN fundamentals
- GROUP BY

Examples:

- Customer filtering
- Product analysis
- Transaction reporting
- Business record retrieval

---

### Joins and Aggregation

Concepts practiced:

- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- Aggregate functions
- Multi-table analysis
- Subqueries
- Business reporting

Examples:

- Corporate client analysis
- Building and apartment management databases
- Supplier and ingredient tracking
- Operations analytics

---

### Advanced SQL

Concepts practiced:

- UNION
- INTERSECT
- Nested subqueries
- Ranking logic
- Multi-table business analytics
- Category averages
- Sales analysis
- Advanced reporting queries

Examples:

- Product ranking systems
- Store performance analytics
- Vendor sales reporting
- Zip code market analysis
